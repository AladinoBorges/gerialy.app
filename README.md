## Getting Started

First, install dependencies and run the development server using **Bun**:

```bash
bun install
bun dev
```

Other package managers can be used as fallback:

```bash
# npm
npm install
npm run dev

# pnpm
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
